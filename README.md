## Tips
- Recommended photo formats: .jpg or .webp (smaller file size)
- Profile photo: ideally square or portrait orientation
- Art images: landscape orientation works best (4:3 ratio)
- Compress images before uploading at https://squoosh.app
  (keeps the site fast — aim for under 300KB per image)

## How to link an image in the gallery
Find the art card in index.html and add an <img> tag inside .art-img:

  <div class="art-img art-color-1">
    <img src="images/iit-kgp-2024.jpg" alt="InterIIT 2024 Canvas">
    <!-- delete the .art-mock div below once you add the image -->
  </div>
